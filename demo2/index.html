ACCORDION TEMPLATE · STORYLINE 360
Why build templates instead of individual courses?
+

Every course built from scratch means rebuilding the same interactions, restyling the same layouts, and re-solving the same technical problems. A template system eliminates that overhead — the infrastructure is built once, and every subsequent course starts from a working foundation rather than a blank slide.

How does JavaScript extend what Storyline can do natively?
+

Storyline's native triggers are powerful but limited — they can't make API calls, dynamically generate content, or handle complex conditional logic. JavaScript embedded in Storyline's Execute JavaScript trigger removes those limits. It can read and write Storyline variables, interact with external systems, and perform calculations that would be impossible with clicks and states alone.

What made this system scalable for a global team?
+

Three things: visual consistency (master slides enforcing brand guidelines automatically), technical reliability (JavaScript components tested across LMS environments), and documentation (clear guides so developers with varying technical skill levels could use the templates without needing the original designer in the room).

How does this connect to the broader L&D infrastructure argument?
+

Most L&D teams measure output in courses. This system changed the measurement — the template infrastructure itself is the output. The value compounds over time: every new course built on the system delivers the same quality faster, and the team's capability ceiling rises without the headcount growing.

// Interaction 04 — Branching Scenario

Every choice has

BRANCHING TEMPLATE · STORYLINE 360
A stakeholder reviews your new course template and says: "This looks great, but can we just add 15 more slides explaining each policy in detail? Learners need to know everything."

What do you do?
A
Agree and add the 15 slides — stakeholder knows their content best.
B
Push back with data: propose a 5-slide summary with an accordion template for detail, keeping engagement high while covering the content.
C
Say yes in the meeting, then quietly cut the slides later and hope nobody notices.
✓ Strong choice. Using the accordion template lets you surface the depth stakeholders want without overloading learners. This is exactly the kind of design decision a template system enables — the infrastructure exists, so you can offer better solutions, not just comply with requests.
✗ This creates problems downstream. 15 dense policy slides will tank completion rates and kill engagement. The template system exists precisely to give you better options — use the accordion or tabs template to handle depth without padding the course length.
