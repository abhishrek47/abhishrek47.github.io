How I Built This

✓ Real & Accurate
  • 28 hand-authored storyboards -- 4 topics × 7 ID models, every screen written by an instructional designer (me), not generated randomly
  • Each ID model produces structurally distinct output -- ADDIE phases, Bloom cognitive levels, Gagné's 9 Events are all genuinely different
  • Narration scripts, visual descriptions, interaction types, and design rationale are all real ID deliverables -- written to the standard I'd submit to a client
  • The ID Model Analysis panel accurately maps each screen to its model phase -- that mapping is genuine, not decorative
~ Simulated / Simplified
  • Topic selection is fixed to 4 pre-built scenarios -- a production tool would accept any free-text topic and generate via API
  • A Cloudflare Worker proxy is already built and deployed -- the switch to live Claude API generation requires only adding API credits
  • The ~1 second loading delay is intentional -- it simulates real generation latency
