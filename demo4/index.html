How I Built This

✓ What's real in this demo
  • Every xAPI statement is built to the full xAPI 1.0.3 specification -- real actor IRI, real verb IRI from the ADL registry
  • The 4-screen course structure mirrors an actual LMS-delivered module
  • Analytics section reflects real statement data generated by your specific interactions
~ What's simulated
  • No real LRS connected -- statements generated in-memory in JavaScript
  • Video player is a styled simulation -- progress fires on a timer
  • Learner identity (Alex Chen) is a fixed simulation
